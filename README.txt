# Portal Ilha da Gigóia (HTML/CSS/JS)

## Como rodar
Abra o arquivo `index.html` no navegador (ou use um servidor local).

## Estrutura
- index.html (Home)
- a-ilha.html
- passeios-rotas.html
- comer-beber.html
- hospedagem.html
- o-que-fazer.html
- planeje.html (Monte seu roteiro + modal de telefone)
- styles.css (estilos)
- app.js (comportamento / conteúdo dinâmico)
- assets/ (imagens)

## Imagens
Os arquivos em `assets/` estão como placeholders vazios.
Substitua pelos seus arquivos de imagem (mesmos nomes).
